## Généralités

L’outil **« Trouver une adresse »** permet de trouver l’adresse, la parcelle ou le toponyme d’un lieu sur la carte, et de copier le résultat.

Vérifiez au préalable que l’outil est bien sélectionné dans le [catalogue d’outils](https://cartes.gouv.fr/aide/fr/guides-utilisateur/visualiseur-cartographique/catalogue-d-outils/).

Cliquez sur l’icône de l’outil :

![Image décrivant le panneau de l’outil trouver une adresse](/img/guides-utilisateur/visualiseur-cartographique/les-outils-autour-de-la-carte/trouver-une-adresse/01_trouver-une-adresse.png){.fr-responsive-img .frx-border-img .frx-img-contained}

Le panneau de l’outil s’ouvre alors :

![Image décrivant le panneau de l’outil trouver une adresse](/img/guides-utilisateur/visualiseur-cartographique/les-outils-autour-de-la-carte/trouver-une-adresse/02_panneau-trouver-une-adresse.png){.fr-responsive-img .frx-border-img .frx-img-contained}

---

## Adresses, toponymes ou parcelles ?

Choisissez dans la liste ce que vous voulez rechercher :

![Image décrivant le choix de ce qu’on souhaite rechercher](/img/guides-utilisateur/visualiseur-cartographique/les-outils-autour-de-la-carte/trouver-une-adresse/03_nature-de-la-recherche.png){.fr-responsive-img .frx-border-img .frx-img-contained}

Vous avez le choix entre **« Adresses »**, **« Lieux/toponymes »** ou **« Parcelles cadastrales »**.

---

## Où rechercher ?

Choisissez dans la liste le périmètre de votre recherche :

![Image décrivant à quel endroit on peut rechercher](/img/guides-utilisateur/visualiseur-cartographique/les-outils-autour-de-la-carte/trouver-une-adresse/04_comment-rechercher.png){.fr-responsive-img .frx-border-img .frx-img-contained}

Vous avez le choix entre **« Pointer un lieu »**, **« Dessiner un cercle »** ou **« Dessiner une emprise »**.

### Pointer un lieu

Une fois l’option sélectionnée, cliquez une fois sur la carte pour choisir un point :

![Image décrivant comment rechercher avec un point](/img/guides-utilisateur/visualiseur-cartographique/les-outils-autour-de-la-carte/trouver-une-adresse/05_rechercher-avec-un-point.png){.fr-responsive-img .frx-border-img .frx-img-contained}

Cliquez sur **« Rechercher »**. L’outil affichera alors le résultat le plus proche :

![Image décrivant le résultat avec un point](/img/guides-utilisateur/visualiseur-cartographique/les-outils-autour-de-la-carte/trouver-une-adresse/06_resultat-avec-un-point.png){.fr-responsive-img .frx-border-img .frx-img-contained}

### Dessiner un cercle

Une fois l’option sélectionnée, cliquez une première fois sur la carte pour choisir le centre du cercle, puis une seconde fois pour choisir la taille du cercle :

![Image décrivant comment rechercher avec un cercle](/img/guides-utilisateur/visualiseur-cartographique/les-outils-autour-de-la-carte/trouver-une-adresse/07_rechercher-avec-un-cercle.png){.fr-responsive-img .frx-border-img .frx-img-contained}

Cliquez sur **« Rechercher »**. L’outil affichera alors les résultats dans le cercle (avec une limite maximale de 20 résultats) :

![Image décrivant le résultat avec un cercle](/img/guides-utilisateur/visualiseur-cartographique/les-outils-autour-de-la-carte/trouver-une-adresse/08_resultat-avec-un-cercle.png){.fr-responsive-img .frx-border-img .frx-img-contained}

### Dessiner une emprise

Une fois l’option sélectionnée, cliquez une première fois sur la carte pour choisir une extrémité de l’emprise, puis une seconde fois pour choisir l’autre extrémité :

![Image décrivant comment rechercher avec une emprise](/img/guides-utilisateur/visualiseur-cartographique/les-outils-autour-de-la-carte/trouver-une-adresse/09_rechercher-avec-une-emprise.png){.fr-responsive-img .frx-border-img .frx-img-contained}

Cliquez sur **« Rechercher »**. L’outil affichera alors les résultats dans l’emprise (avec une limite maximale de 20 résultats) :

![Image décrivant le résultat avec une emprise](/img/guides-utilisateur/visualiseur-cartographique/les-outils-autour-de-la-carte/trouver-une-adresse/10_resultat-avec-une-emprise.png){.fr-responsive-img .frx-border-img .frx-img-contained}

---

## Copier le résultat

Vous pouvez copier chaque résultat de la recherche en cliquant l’icône **« Copier le résultat »** en face du résultat à copier :

![Image décrivant comment copier un résultat](/img/guides-utilisateur/visualiseur-cartographique/les-outils-autour-de-la-carte/trouver-une-adresse/11_copier-le-resultat.png){.fr-responsive-img .frx-border-img .frx-img-contained}

---

## Modifier une recherche existante

Si vous êtes sorti de l’outil **« Trouver une adresse »** et que vous souhaitez faire une nouvelle recherche, cliquez sur l’icône **« Trouver une adresse »** pour afficher les résultats précédents.

Cliquez alors sur le bouton **« Retour »** pour revenir à l’étape précédente et modifier les options ou le lieu de recherche.
