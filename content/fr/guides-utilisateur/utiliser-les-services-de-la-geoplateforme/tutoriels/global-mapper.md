## Introduction


[*<span lang="en">Global Mapper</span>*](http://www.globalmapper.fr/){target="_blank" rel="noopener noreferrer" title="Global Mapper"} est une application logicielle SIG de pointe, disponible en version française et dont [*<span lang="en">Géom@tique</span>*](http://www.geomatique.fr/){target="_blank" rel="noopener noreferrer" title="Géom@tique"} est le distributeur exclusif pour la francophonie, qui fournit aux professionnels de la géomatique, novices ou expérimentés, une gamme complète d'outils de visualisation, d'édition et d'analyse de données spatiales. Si Global Mapper prend en charge une liste inégalée de formats de données, il permet également d’utiliser les services web délivrés par la Géoplateforme (WMS, WMTS et WFS) ainsi que, dans sa version « Pro », d’exploiter les données [*<span lang="en">LiDAR HD</span>*](https://cartes.gouv.fr/aide/fr/partenaires/ign/observations-regulieres-territoire/relief/nuages-points-lidar-hd/){target="_blank" rel="noopener noreferrer" title="LiDAR HD"} (édition, classification, extraction vectorielle, etc.).

## Tutoriel


[*<span lang="en">Géom@tique</span>*](http://www.geomatique.fr/){target="_blank" rel="noopener noreferrer" title="Géom@tique"} vous propose un [*<span lang="en">tutoriel Global Mapper</span>*](https://data.geopf.fr/annexes/ressources/documentation/Tutoriel_Global_Mapper.pdf){target="_blank" rel="noopener noreferrer" title="Tutoriel Global Mapper"} en trois volets :

- Importer les données IGN intégrées
- Importer des données en ligne personnalisées
- Importer des données LiDAR HD
 
Le tutoriel est disponible en [*<span lang="en">version PDF</span>*](https://data.geopf.fr/annexes/ressources/documentation/Tutoriel_Global_Mapper.pdf){target="_blank" rel="noopener noreferrer" title="Tutoriel Global Mapper PDF"} et vous pouvez également le retrouver dans la rubrique FAQ du forum géomatique.fr (inscription nécessaire).
